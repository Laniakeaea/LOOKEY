## English

A keyboard and mouse input visualization overlay for Windows.

![Platform](https://img.shields.io/badge/platform-Windows-blue)
![License](https://img.shields.io/badge/license-MIT-green)

### Overview

Keymera captures and displays keyboard and mouse inputs as a floating overlay window. Designed for content creators, streamers, educators, and anyone who needs to demonstrate input interactions.

### Features

- Real-time keyboard key visualization with press/release states
- Mouse button and scroll wheel status display
- Draggable overlay window with adjustable position
- System tray integration for quick access
- Multiple theme support (dark/light)
- Customizable scale (50%-200%)
- Startup with Windows option

### System Tray Menu

Right-click the system tray icon to access settings:

| Menu | Description |
|------|-------------|
| **Anchor** | Set overlay position (top-left, top-right, bottom-left, bottom-right) |
| **History Direction** | Choose key history display direction (up/down) |
| **Theme** | Switch between dark and light themes |
| **Scale** | Adjust overlay size (50%, 75%, 100%, 125%, 150%, 175%, 200%) |
| **Startup with Windows** | Enable/disable auto-start on login |
| **Mouse Side Buttons** | Enable/disable mouse side button tracking |
| **Drag Mode** | Toggle overlay draggability |
| **Adjust Current/History Alpha** | Fine-tune opacity for current and historical keys |
| **Factory Reset** | Restore default settings |
| **Exit** | Close the application |

## 中文

一款适用于 Windows 的键盘鼠标输入可视化覆盖窗口。

![平台](https://img.shields.io/badge/platform-Windows-blue)
![许可证](https://img.shields.io/badge/license-MIT-green)

### 简介

Keymera 将键盘和鼠标的输入实时显示为浮动覆盖窗口。专为内容创作者、直播主播、教育工作者以及需要演示输入操作的用户设计。

### 功能特性

- 实时键盘按键可视化，支持按下/释放状态显示
- 鼠标按键及滚轮状态展示
- 可拖拽的浮动窗口，位置可调节
- 系统托盘集成，快速访问设置
- 多主题支持（深色/浅色）
- 可调缩放比例（50%-200%）
- 开机自启动选项

### 系统托盘菜单

右键点击系统托盘图标访问设置：

| 菜单项 | 说明 |
|--------|------|
| **锚点位置** | 设置覆盖窗口位置（左上、右上、左下、右下） |
| **历史方向** | 选择按键历史的显示方向（向上/向下） |
| **主题** | 在深色和浅色主题间切换 |
| **缩放** | 调整覆盖窗口大小（50%、75%、100%、125%、150%、175%、200%） |
| **开机自启** | 启用/禁用登录时自动启动 |
| **鼠标侧键** | 启用/禁用鼠标侧键追踪 |
| **拖拽模式** | 切换覆盖窗口可拖拽性 |
| **调整当前/历史透明度** | 精细调节当前按键和历史按键的不透明度 |
| **恢复出厂设置** | 恢复默认设置 |
| **退出** | 关闭应用程序 |
